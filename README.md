This repository contains a simple Activity Tracker application built using HTML, CSS, and JavaScript. It is designed for beginners to understand how to make a static webpage interactive.
Users can view a list of activities and mark them as completed. When an activity is updated, the changes are reflected instantly on the screen without reloading the page. The project also shows real-time progress and stores data using browser Local Storage.
This project is useful for students who want to learn:
1.Basic JavaScript concepts
2.DOM manipulation
3.Event handling
4.Dynamic UI updates
It is a beginner-friendly project and can be easily extended with more features like adding or deleting tasks.
